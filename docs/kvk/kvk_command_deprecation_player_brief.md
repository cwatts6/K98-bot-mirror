# KVK Command Update Briefing

Phase 7 redirect behaviour has been tested successfully. Share after the Phase 7 deprecation
redirect PR is deployed.

## Discord Message

KVK commands have moved into one home: `/kvk`.

The old command names now show a short redirect message and will be removed after the feedback
window. Please start using the new commands below.

What changed:

- `/mykvkstats` is now `/kvk stats`
- `/mykvktargets` is now `/kvk targets`
- `/mykvkhistory` is now `/kvk history`
- `/kvk_rankings` is now `/kvk rankings type:kvk`
- `/honor_rankings` is now `/kvk rankings type:honor`
- `/prekvk report` is now `/kvk rankings type:prekvk` in the KVK stats channel

Channel guide:

- `/kvk targets` opens in the KVK targets channel.
- `/kvk stats`, `/kvk history`, and `/kvk rankings` open in the KVK stats channel.
- Admins can use the new KVK commands outside those channels when needed.

What is better in the new commands:

- `/kvk stats` shows the modern personal KVK stats card.
- `/kvk targets` shows the modern target progress card.
- `/kvk history` includes History, Summary, Trends, and CSV export controls.
- `/kvk rankings` brings KVK, Honor, PreKvK, and Hall of Fame records into one browser.
- Ranking views include visual Top 10 cards, Top 25/50 compact views, private My Rank, and CSV
  export where supported.

Nothing has changed about your registered accounts or KVK data. Only the command names are being
tidied up.

When using Discord, type `/kvk` and pick the action you need from the subcommand list.
